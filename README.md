## Feature
- [X] List daily news top headlines.
- [X] Filter daily news by category.
- [X] Refresh list daily news with pull to refresh style.
- [X] Go to detail news website.
- [X] Dark mode support.
- [X] Firebase support.
- [X] Hive Database.
- [X] Authenticate using Gmail.
